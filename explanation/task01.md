# task01

Что выведет программа?
Объясните вывод программы.

```go
package main

import "fmt"

func main() {
  a := [5]int{76, 77, 78, 79, 80}
  var b []int = a[1:4]
  fmt.Println(b)
}
```


## Вывод:
```
[77 78 79]
```

## Объяснение

Оператор a[1:4] берёт срез элементов с индексом 1 (включительно) до 4 (исключительно) от массива а.
